# Anime.Directory
Anime.Directory is a website to check out anime before watching. See top anime, what’s hot this season, upcoming shows, or movies. Click a card to see the score, studio, genres, synopsis, and a trailer.

Made with just HTML, CSS, and JavaScript. No frameworks, no setup, nothing fancy.

## Features
Browse Top Anime, This Season, Upcoming, and Movies
Live search with trending shortcuts
Click a show for details and trailer
Handles API limits automatically
Retry button if something fails
